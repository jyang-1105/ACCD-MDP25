Cardboard DJ

ESP32 Based Motion Music Controller

1. About the Project

This is a DIY MIDI controller built using recycled cardboard and electronic components. I used an ESP32 board to read analog sensors (joystick and potentiometer) and digital inputs (buttons), sending the data to a computer via serial communication. A custom program written in Processing receives this data and plays corresponding audio samples in real-time, effectively turning a cardboard box into a functional drum machine.

2. Working Principle

The system architecture consists of four layers:

  • Input Layer: The joystick and potentiometer act as variable resistors, outputting voltage between 0V and 3.3V. The button sends a binary high/low signal.
  • Processing Layer (ESP32): The microcontroller's ADC converts the analog voltage into digital values ranging from 0 to 4095.
  • Communication Layer: The ESP32 transmits data to the computer via USB Serial at a baud rate of 115200. The data format is a string: X, Y, Button, Knob.
  • Software Layer: The computer program parses the string. To ensure accuracy, I implemented a Deadzone Algorithm. Sound is only triggered when joystick values exceed specific thresholds (< 500 or > 3600), preventing accidental triggers due to sensor drift.

3. Controls & Functions

The mapping between hardware inputs and sound effects is as follows:

Joystick (4-Way Trigger)
  • UP: Closed Hi-Hat
  • DOWN: Open Hi-Hat
  • LEFT: Synth FX Sound
  • RIGHT: Crash Cymbal
Red Button Kick Drum (The main beat)
Blue Knob Master Volume Control
