Infrared Sensor
Infrared Sensing Point
Continuously polls the surrounding area at 30 Hz. When a human enters the detection radius, it emits a proximity signal to the microcontroller. This single input governs all downstream behaviour.
Input
Proximity signal
02
Status Display
Miniscope LCD
A small LCD screen on the machine body showing real-time sensor values — distance readings and trigger confirmation. Acts as a system diagnostic display, not a user-facing output.
Feedback
Confirmed trigger
03
Microcontroller
Mainframe Inside
Central logic unit. Reads the proximity signal and switches between idle and triggered states. When triggered, it dispatches commands to both output channels simultaneously: the display screen and the thermal printer.
Process
Parallel outputs — both channels fire simultaneously
